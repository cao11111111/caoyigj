# 设计指南 - 曹一工具箱

## 品牌定位

- **产品名称**：曹一工具箱
- **产品类型**：校园智能工具小程序
- **设计风格**：现代简洁、专业可靠、校园友好
- **目标用户**：学校师生
- **使用场景**：知识卡片生成、学习工具、日常助手

## 配色方案

### 主色调
- **主色（Primary）**：`#2563EB` (Blue 600)
- **主色悬停**：`#1D4ED8` (Blue 700)
- **主色浅色**：`#EFF6FF` (Blue 50)

### 辅助色
- **成功色**：`#10B981` (Emerald 500)
- **警告色**：`#F59E0B` (Amber 500)
- **错误色**：`#EF4444` (Red 500)
- **紫色点缀**：`#8B5CF6` (Purple 500)

### 中性色
- **标题文字**：`#1E293B` (Slate 800)
- **正文文字**：`#475569` (Slate 600)
- **次要文字**：`#94A3B8` (Slate 400)
- **占位符文字**：`#CBD5E1` (Slate 300)
- **分割线**：`#F1F5F9` (Slate 100)

### 背景色
- **页面背景**：`#F8FAFC` (Slate 50)
- **组件背景**：`#FFFFFF` (White)

## 字体规范

- **大标题**：24px，font-weight: 700
- **标题**：18-20px，font-weight: 600
- **正文**：14-16px，font-weight: 400
- **辅助文字**：12px，font-weight: 400

## 间距系统

- **页面边距**：16px
- **卡片间距**：12-16px
- **组件内边距**：16-20px
- **列表项间距**：8-12px

## 圆角系统

- **小圆角**：8px（按钮、标签）
- **中圆角**：12px（输入框、小卡片）
- **大圆角**：16px（主卡片、面板）
- **全圆角**：9999px（头像、胶囊）

## 阴影系统

- **卡片阴影**：`0 2px 8px rgba(0, 0, 0, 0.08)`
- **悬浮阴影**：`0 4px 12px rgba(0, 0, 0, 0.12)`

## 组件使用原则

**通用 UI 组件优先使用 `@/components/ui/*`**
- 按钮 → `Button`
- 输入框 → `Input` / `Textarea`
- 卡片 → `Card`
- 头像 → `Avatar`
- 列表 → `View` + Tailwind
- 标签 → `Badge`
- 弹窗 → `Dialog`

**禁止用 View/Text 手搓通用组件**

## 页面结构

### 登录页
- Logo + 应用名称居中
- Tab 切换（账号登录 / 微信登录）
- 输入框 + 登录按钮
- 底部隐私协议

### 完善信息页
- 标题 + 说明
- 输入框列表（昵称、学校、科目）
- 提交按钮

### 首页（智能体）
- 顶部欢迎卡片（渐变蓝色背景）
- 用户信息展示（头像 + 昵称）
- 快捷入口网格（2x2，图标+标题）
- 最近对话列表

### 历史记录
- 页面标题
- 按日期分组（今天/昨天/更早）
- 对话卡片列表

### 个人中心
- 用户头像卡片
- 统计数据（对话次数、累计使用等）
- 功能菜单列表（设置、关于等）

### 知识卡片
- 输入区域（标题 + 内容）
- 生成按钮
- 结果展示页面

- **我的**：User 图标

- 未选中色：`#94A3B8`
